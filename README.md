# emergency-response-simulator
A Java OOP simulation of dispatching units across a grid to handle emergencies.

## Tech Stack
- Language: Java
- Tools/Frameworks: Maven

## Overview
The simulation models a city grid where emergency incidents are generated and response units are dispatched to handle them. Units are managed and tracked across the grid, with the simulation running across multiple ticks to reflect real-time emergency response.

## Features
- Polymorphism to model multiple emergency unit types 
- Determinism 
- Turn-based simulation across multiple ticks 

## How to Run
Open the project in your preferred Java IDE and build and run the tests using Maven.

## What I Learned
This project solidified my understanding of core OOP principles including inheritance, polymorphism, and encapsulation, applying them to a real simulation problem. Working with Maven also introduced me to professional Java development workflows including dependency management and unit testing.
